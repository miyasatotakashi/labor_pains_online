# README
## 開発用語
- Ruby 3.0.1
- Rails 6.1.7
## 就業Termの技術
- devise
- AWS EC2
## カリキュラム外の技術
- LINE　ログイン
- LINE BOT 連携
## 実行手順
$ git clone git@github.com:miyasatotakashi/labor_pains_online.git

$ cd labor_pains_online

$ bundle install

$ rails db:create && rails db:migrate

$ rails s
